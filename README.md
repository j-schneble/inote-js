# iNote

>Once an account has been registered and succesfully logged in, the user can then create notes.  
All notes will display on the dashboard and recieve a timestamp.  
Notes can be edited or deleted, and will remain on the dashboard unless deleted by the user.

- [MongoDb](https://www.mongodb.com/)

- [Node.js](https://nodejs.org/en/)
